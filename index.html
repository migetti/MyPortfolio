Candy's Make up

Candys's Make up is a responsive website designed for a beauty salon entrepreneur in need for a new online presence.

Web Design

Nov 2015

Project brief

Candy's Make up approached me to design and develop a website for a starting business

Website Design

Candy's Makeup wanted a design that has a 1920's jazz age look and feel. From the start, Candy's makeup was designed to be fully responsive and cross-browser compatible. The main focus would be to create a image gallery to showcase her previous work to establish trust in the first customer touchpoint:

  1. Headline - "making my clients look wonderful using the best products and the lates techniques in beauty care"
  2. Navigation - Links to other online content
  3. Branding - Photos about previous works to establish trust
  4. Elements - High quality photos that load quickly, provided services, price list, location
  5. Call to action - A button to for booking an appointment through a 3rd party booking system
